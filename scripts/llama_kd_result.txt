# === JUDGMENT (honest) ===
# KD HELPED, clearly and in the predicted way. The gap to fp narrowed from 263x (hard-CE) to 146x
# (KD), a 1.80x improvement. More importantly the FAILURE MODE changed: hard-CE OVERFIT (best at
# epoch 1, then monotonically WORSE on held-out while train loss fell), whereas KD's held-out ppl
# decreased MONOTONICALLY every epoch (2679 -> 1279) and was STILL FALLING at epoch 5 -- the soft
# teacher targets regularize the tiny corpus that the one-hot labels overfit. The model is now
# COMPUTE/EPOCH-bound (not yet plateaued) AND still DATA-bound (4303 train tokens is tiny); KD did
# not reach fp (146x is far from 1x) -- consistent with the standing finding that a real ternary
# conversion needs a real-scale corpus + longer schedule. The host full-vocab (49152) softmax is the
# wall-time bottleneck (one-time teacher cache = 42 fp forwards + the per-epoch reductions). HONEST:
# KD is a genuine, measurable lever on the conversion gap; it did not close it on this small set.
