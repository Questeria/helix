Autodiff, in the language

Gradients are a keyword, not a library.

Forward and reverse-mode autodiff lower into the AST at compile time — no runtime tape, no tracing, no Python overhead. Every transcendental in the standard library participates in the chain rule.

  • grad(f)(x)Forward-mode derivative. Symbolic, fully inlined.
  • grad_rev_all(f)(...)Reverse-mode adjoints; returns a struct of all partials.
  • @checkpointRematerialize segments instead of caching activations.
  • @kernelEmit the same function as a PTX kernel for the GPU.
